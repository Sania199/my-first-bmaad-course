# Introduction to Microsoft Excel for Beginners
## 2-Hour Course Outline

---

### **Topic 1: Excel Basics & Navigation** ⏱️ 25 minutes
- Welcome and course overview (3 min)
- Opening Excel and understanding the interface
  - The ribbon, toolbar, and menu options
  - Workbooks, worksheets, and cells
- Basic terminology: rows, columns, cells, and cell references
- Creating and saving your first workbook
- **Activity**: Open Excel and create a blank workbook

---

### **Topic 2: Data Entry & Formatting** ⏱️ 30 minutes
- Entering different types of data (text, numbers, dates)
- Basic cell formatting
  - Font styles and sizes
  - Cell colors and borders
  - Number formats (currency, percentages, decimals)
- Column width and row height adjustment
- Auto-fill and copying data
- **Activity**: Create a sample budget table with formatted data

---

### **Topic 3: Simple Formulas & Functions** ⏱️ 35 minutes
- Introduction to formulas (syntax and basics)
- Basic arithmetic operations (+, -, *, /)
- Essential functions for beginners:
  - SUM (adding numbers)
  - AVERAGE (calculating averages)
  - COUNT (counting cells)
  - MIN/MAX (finding smallest/largest values)
- Copying formulas and absolute vs. relative references
- **Activity**: Create a sales report with SUM and AVERAGE calculations

---

### **Topic 4: Charts & Summary** ⏱️ 25 minutes
- Why visualize data with charts
- Creating your first chart
  - Column charts
  - Pie charts
  - Line charts
- Chart elements: titles, legends, and labels
- Basic chart customization
- Tips for presenting data effectively
- **Activity**: Create a chart from the sales data and customize it
- **Bonus**: Review key shortcuts and best practices

---

## **Course Summary**
By the end of this 2-hour course, beginners will be able to:
✓ Navigate Excel confidently  
✓ Enter and format data  
✓ Create basic calculations with formulas  
✓ Visualize data with charts  

---

**Total Duration**: 2 hours (115 minutes of instruction + 5 minutes for breaks/transitions)
